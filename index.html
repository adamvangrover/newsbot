Platform Capabilities

Built to bridge the gap between raw data and actionable intelligence.

Synthetic Data Engine

Generate complex, realistic financial datasets including market events, corporate actions, and news sentiment using the SyntheticDataEngine. Outputs directly to JSONL and Parquet for ML training.

Impact Analysis

Trace second and third-order effects of market events using the ReasoningCore. Visualize how a supplier delay ripples through the supply chain to affect stock prices.

Modern React Dashboard

A full-featured SPA built with React, Vite, and Tailwind CSS. Includes a Repository Explorer, Signal Intelligence feeds, and an Operational Dashboard for real-time monitoring.

System Architecture

Backend Core (FastAPI)

Handles API requests, orchestrates the synthetic data generation, and serves the impact analysis engine logic. Located in app/.

2

Frontend (React + Vite)

A static-deployable SPA that consumes data from the backend or static JSON files. Features a Repo Explorer and dynamic dashboards. Located in frontend/.

3

Semantic Narrative Library

The brain of the operation. Contains the reasoning engines, graph algorithms, and scenario simulators. Located in semantic_narrative_library/.

Synthetic Data Generation

The `SyntheticDataEngine` is capable of creating vast amounts of realistic financial data. It doesn't just randomize numbers; it builds a causal graph of events. If a "Strike" event occurs in the "Logistics" sector, the engine generates downstream "Delay" events for companies that rely on that sector, accurately modeling supply chain disruptions.

Operational Dashboard

The frontend dashboard visualizes the data streams. It features a "Matrix View" for high-density information display and an "Impact Graph" that renders the relationships between entities. It can run in a "Static Mode" consuming the generated JSON files, making it perfect for demonstrations without a live backend.
